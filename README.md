# Project Title
Gabut

## Description
Gabut is a project that aims to provide tools and utilities to enhance productivity and collaboration among users.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/heosjxhao/gabut.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gabut
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage Guide
To start the application, use the following command:
```bash
npm start
```
